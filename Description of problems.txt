Преша помилка виникає у тесті test_add_existing_child

Якщо коротко описувати опис проблеми, то можна сказати що наш тест очікує, що при додаванні учня, який вже є в нашому словнику
буде підніматись виключення ValueError, але цього не відбувається.

Розповідаючи про вирішення цієї проблеми, то нам потрібно додати перевірку у функцію add_child, я б зробив це ось так:
def add_child(data_dict, child):
    if child in data_dict:
        raise ValueError("Child already exists")
    data_dict[child] = {}
    return len(data_dict)
